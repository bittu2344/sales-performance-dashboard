# 📊 Sales Performance Dashboard

**Author:** Bittu Kumar | Data Analyst
**Tools:** Python | Pandas | SQL | Matplotlib | Seaborn | Power BI | Excel

## 🎯 Project Overview
Analyzed 12 months of retail sales data (10,000+ records) across regions, products, and sales reps. Built a full dashboard revealing a **23% revenue gap** between top and bottom regions.

## 🔍 Key Findings
- North region outperforms South by ~23% in revenue
- Laptops are the highest revenue-generating product category
- Month-over-Month growth was strongest in Q3

## ⚙️ How to Run
1. Install: `pip install pandas numpy matplotlib seaborn`
2. Run directly — no dataset download needed!
   `python sales_dashboard.py`
